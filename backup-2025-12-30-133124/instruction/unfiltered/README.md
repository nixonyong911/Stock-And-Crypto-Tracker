# Unfiltered Session Logs

This folder contains raw, unfiltered session documentation from AI-assisted development sessions.

## Purpose

- **Immediate capture**: Document everything done in a session before organizing
- **Raw details**: Include errors, fixes, debugging steps, and learnings
- **Future reference**: Source material for creating polished documentation

## Workflow

1. **During session**: AI documents work in `unfiltered/` with date prefix
2. **After session**: Review and extract key information
3. **Organize**: Move relevant content to appropriate folders:
   - Architecture docs → `instruction/architecture/`
   - Completed tasks → `instruction/completed/`
   - CLI commands → `instruction/cli/`
   - Runbooks → `instruction/runbooks/`
4. **Archive or delete**: Old unfiltered docs can be archived or removed

## Naming Convention

```
YYYY-MM-DD-<short-description>.md
```

Example: `2025-12-30-twelvedata-backoffice-integration-session.md`
